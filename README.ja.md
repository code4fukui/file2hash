# file2hash

ファイルのSHA-512、SHA-256、SHAKE-128、MD5ハッシュ値を計算するWebアプリケーションです。

## 機能
- SHA-512、SHA-256、SHAKE-128、MD5のハッシュ値を計算
- ハッシュ値をBase32およびBase16(16進数)で表示
- ファイルサイズの表示
- ファイルをドラッグ&ドロップで入力可能

## 使い方
1. [デモサイト](https://code4fukui.github.io/file2hash/)にアクセス
2. ファイルをページにドラッグ&ドロップ
3. ファイルのハッシュ値が即座に表示される
