# file2hash

ファイルのハッシュ値を計算するWebアプリケーションです。SHA-512、SHA-256、SHAKE-128、MD5のハッシュ値を表示できます。

## デモ
https://code4fukui.github.io/file2hash/

## 機能
- SHA-512、SHA-256、SHAKE-128、MD5のハッシュ値を計算
- Base32とBase16(16進数)でのハッシュ値表示
- ファイルサイズの表示
- ファイルをドラッグ&ドロップできる

## 使い方
1. [デモサイト](https://code4fukui.github.io/file2hash/)にアクセスする
2. ファイルをページにドラッグ&ドロップする
3. ファイルのハッシュ値が即座に表示される
